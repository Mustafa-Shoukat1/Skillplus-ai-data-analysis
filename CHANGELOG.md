# Changelog

All notable changes to SkillsPulse will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-03

### Added
- Initial release of SkillsPulse AI-Powered Data Analysis Platform
- FastAPI backend with async support
- Next.js 14 frontend with App Router
- LangGraph-based AI analysis workflow
- Support for Claude (Anthropic) and GPT-4 (OpenAI) models
- CSV and Excel file upload support
- Interactive ECharts visualizations
- Role-based authentication (Admin/Viewer)
- User management system
- Analysis visibility controls
- Comprehensive API documentation (Swagger/ReDoc)

### Backend Features
- RESTful API with FastAPI
- SQLite database with SQLAlchemy ORM
- JWT-based authentication
- File upload with validation
- Background task processing
- Comprehensive logging system

### Frontend Features
- Modern UI with Tailwind CSS and Radix UI
- Admin dashboard with file upload and analysis
- Viewer dashboard for public analyses
- Real-time analysis status tracking
- Chart download functionality
- Responsive design

### AI Analysis Capabilities
- Natural language query processing
- Automatic query classification
- Data extraction and transformation
- Dynamic chart generation
- Code review and optimization
- Error handling and retry logic

## [Unreleased]
